MOF DISCORD BOT

Getting Started

  1. Check for MOF BOT on the right side bar of our discord server.

  2. Send a DM(private message) !start to the Bot

    You will be giving a new private key and a public key.You are responsible to keep your private key safe, we do not store your private keys.

Command List

1

!help
this command displays all the commands available and a brief description about the commands

2

!wallet
this command displays your address that is been tracked. to set an address to be tracked use !set_wallet , note this is different from what you get from !start and !restore

3

!start
use this command to get started,a new account details(private key and wallet address) will be generated for your personal this, this will also enable you to use the !buy and !sell commands

4

!orderbook
this command is used to check and display all(buy and sell orders) market orders of a token by using the token market ID

5

!set_wallet
this command is used to set address to be tracked; tracked address can get notifications on new transactions,check tokens held and the worth in $(USD)

6

!restore
this command is used to restore a previous account by private key. note this will over-ride account generated when you ran !start on BOT first launch

7

!start
this command is used to display all the token held by your personal address which was set via !set_wallet ,the token current values in $(USD) will be added soon

8

!check
this command is used to check a token info by using the token mint address

9

!transaction
this command is used to display all your last 10 transactions of your personal wallet which was set via !set_wallet

10

!profile
this command displays a mini profile of all social media accounts where your signed in, balance,private key,addresses,etc...

11

!check_market
this command is used to check a token dex price by using the token market id

12

!buy (coming soon)
this command displays all the commands available and a breif description about the commands

13

!sell (coming soon....)
this command will be used to place a sell order on DEX

14

the bot listens for new transactions and notifies you (coming soon....)

Getting Help

  1. send !help to MOF BOT in a private message.

  2. You can also check for details of a particular in the bot by sending !help the_command_you_need_help_on
